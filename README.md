+ 사이트의 회원들이 판매하고자 하는 상품을 올리거나 사이트에 올라온 상품을 구입할 수 있도록 돕는 것을 목적으로 함.
+ 당근마켓과 번개장터를 벤치마킹 함.
+ 프로젝트 기간 : 05/13 ~ 06/10


## 주요 기능
+ 로그인, 회원가입
+ 판매글 작성 / 검색 / 정렬 / 필터링
+ 개인 정보 관리
+ 채팅
